## Manage Requirements

**ReQuest** helps you efficiently handle and organize your software requirements. It provides comprehensive multiple tools and functionalities to streamline the requirement management process.

### Create Projects

Start by creating the project. You can create a project by clicking on the `Add Project` button on the top right corner of the screen. You will be prompted to enter the project name, description, deadline and coworkers (if any). Once you have entered the details, click on the `Create` button to create the project.

![New Project](/new-project.png)
*Figure: Create a new project*

The project will be created and added to the tabke of projects. You can click on the project to view the details of the it.
