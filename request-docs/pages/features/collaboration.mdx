## Collaboration

### Full-Featured Markdown Editor

The Markdown editor in ReQuest explained on [Using ReQuest to Gather and Analyze Requirements
](/software/request-gath) is a full-featured Markdown editor, and with the help of Hocuspocus
it is a collaborative editor. You can see the changes made by other users in real time.

### Real-time Updates

ReQuest updates every component in real time. You can see the changes made by other users in real time.
This is especially useful when you are working with a team.
